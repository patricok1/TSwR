# TEMAT: Implementacja i analiza algorytmu sterowania MPPI dla symulowanego samochodu

# Cel projektu

Celem jest stworzenie wiarygodnego środowiska symulacyjnego dla pojazdu, opartego na matematycznym modelu dynamiki/kinematyki oraz wykorzystanie algorytmu MPPI do zbadania precyzji i optymalności śledzenia zadanych trajektorii.

# Środowisko z bibliotekami

- PyTorch
- pytorch-mppi
- Numpy
- Scipy
- Matplotlib
